## 系统架构

```
┌─────────────────┐     HTTP/HTTPS      ┌──────────────────┐
│   前端应用       │ ◄─────────────────► │   后端API服务     │
│  (React/Vue)    │                     │  (Flask/FastAPI) │
└─────────────────┘                     └────────┬─────────┘
                                                  │
                                                  │ 调用
                                                  ▼
                                        ┌──────────────────┐
                                        │  模型推理服务     │
                                        │  (PyTorch GPU)   │
                                        └──────────────────┘
```

### 数据流

1. **视频上传模式**：用户上传视频文件 → 后端接收 → 模型推理 → 返回识别结果
2. **实时摄像头模式**：摄像头捕获帧 → 帧缓冲区（48帧）→ 模型推理 → 实时显示结果

## 模型推理服务

创建 `inference.py`：

```python
import torch
import torchvision.transforms as transforms
from PIL import Image
import cv2
import numpy as np
import os
import sys

# 添加模型路径
sys.path.append(os.path.dirname(os.path.abspath(__file__)))
from models.Seq2Seq_v4 import Encoder, Decoder, Seq2Seq_v4


class SignLanguageRecognizer:
    """手语识别器"""
    
    def __init__(self, model_path, device='cuda'):
        """
        初始化识别器
        
        Args:
            model_path: 模型权重文件路径
            device: 计算设备 ('cuda' 或 'cpu')
        """
        self.device = torch.device(device if torch.cuda.is_available() else 'cpu')
        self.sample_duration = 48  # 视频帧数
        self.sample_size = 128     # 图像尺寸
        
        # 加载模型
        self.model = self._load_model(model_path)
        self.model.eval()
        
        # 数据预处理
        self.transform = transforms.Compose([
            transforms.Resize([self.sample_size, self.sample_size]),
            transforms.ToTensor(),
            transforms.Normalize(mean=[0.5], std=[0.5])
        ])
        
        print(f"Model loaded on {self.device}")
    
    def _load_model(self, model_path):
        """加载模型权重"""
        # 模型配置（根据训练时的配置调整）
        enc_hid_dim = 1024
        dec_hid_dim = 1024
        emb_dim = 256
        output_dim = 252  # 词汇表大小
        dropout = 0.5
        
        # 创建模型
        encoder = Encoder(lstm_hidden_size=enc_hid_dim, arch="resnet18")
        decoder = Decoder(
            output_dim=output_dim,
            emb_dim=emb_dim,
            enc_hid_dim=enc_hid_dim,
            dec_hid_dim=dec_hid_dim,
            dropout=dropout
        )
        model = Seq2Seq_v4(encoder=encoder, decoder=decoder, device=self.device)
        
        # 加载权重
        if os.path.exists(model_path):
            model.load_state_dict(torch.load(model_path, map_location=self.device))
            print(f"Loaded model from {model_path}")
        else:
            raise FileNotFoundError(f"Model file not found: {model_path}")
        
        model.to(self.device)
        return model
    
    def preprocess_video(self, video_path):
        """
        预处理视频文件
        
        Args:
            video_path: 视频文件路径
            
        Returns:
            tensor: 预处理后的视频张量 [1, C, T, H, W]
        """
        cap = cv2.VideoCapture(video_path)
        frames = []
        
        # 读取视频帧
        while len(frames) < self.sample_duration:
            ret, frame = cap.read()
            if not ret:
                break
            
            # 转换颜色空间 BGR -> RGB
            frame = cv2.cvtColor(frame, cv2.COLOR_BGR2RGB)
            frame = Image.fromarray(frame)
            frame = self.transform(frame)
            frames.append(frame)
        
        cap.release()
        
        # 如果帧数不足，重复最后一帧
        if len(frames) == 0:
            raise ValueError("No frames extracted from video")
        
        while len(frames) < self.sample_duration:
            frames.append(frames[-1])
        
        # 堆叠成张量 [C, T, H, W]
        frames = torch.stack(frames, dim=1)
        return frames.unsqueeze(0).to(self.device)  # 添加batch维度
    
    def predict(self, video_path):
        """
        对视频进行推理
        
        Args:
            video_path: 视频文件路径
            
        Returns:
            numpy array: 预测的词ID序列
        """
        # 预处理
        input_tensor = self.preprocess_video(video_path)
        
        # 推理
        with torch.no_grad():
            batch_size = 1
            trg_len = 20  # 最大输出长度
            target = torch.zeros(batch_size, trg_len).long().to(self.device)
            target[:, 0] = 1  # SOS token
            
            # 前向传播
            output = self.model(input_tensor, target, teacher_forcing_ratio=0)
            
            # 解码预测结果
            predictions = output.argmax(dim=2).squeeze(1).cpu().numpy()
            
        return predictions
    
    def predict_from_frames(self, frames):
        """
        从帧列表进行推理（用于实时摄像头）
        
        Args:
            frames: 帧列表（PIL Image或numpy array）
            
        Returns:
            numpy array: 预测的词ID序列
        """
        # 预处理帧
        processed_frames = []
        for frame in frames:
            if isinstance(frame, np.ndarray):
                # OpenCV格式 (BGR)
                frame = cv2.cvtColor(frame, cv2.COLOR_BGR2RGB)
                frame = Image.fromarray(frame)
            
            frame = self.transform(frame)
            processed_frames.append(frame)
        
        # 堆叠成张量
        input_tensor = torch.stack(processed_frames, dim=1).unsqueeze(0).to(self.device)
        
        # 推理
        with torch.no_grad():
            batch_size = 1
            trg_len = 20
            target = torch.zeros(batch_size, trg_len).long().to(self.device)
            target[:, 0] = 1
            
            output = self.model(input_tensor, target, teacher_forcing_ratio=0)
            predictions = output.argmax(dim=2).squeeze(1).cpu().numpy()
            
        return predictions


class Vocabulary:
    """词汇表管理"""
    
    def __init__(self, dict_path):
        """
        初始化词汇表
        
        Args:
            dict_path: 字典文件路径
        """
        self.word2idx = {}
        self.idx2word = {}
        self.load_vocab(dict_path)
    
    def load_vocab(self, dict_path):
        """加载词汇表"""
        with open(dict_path, 'r', encoding='utf-8') as f:
            for line in f:
                parts = line.strip().split('\t')
                if len(parts) >= 2:
                    idx = int(parts[0])
                    word = parts[1]
                    self.word2idx[word] = idx
                    self.idx2word[idx] = word
    
    def decode(self, indices):
        """
        将索引序列解码为文本
        
        Args:
            indices: 词ID列表或numpy数组
            
        Returns:
            str: 解码后的文本
        """
        words = []
        for idx in indices:
            if idx in [0, 1, 2]:  # 跳过特殊标记 (PAD, SOS, EOS)
                continue
            if idx in self.idx2word:
                words.append(self.idx2word[idx])
        return ' '.join(words)
```

## API服务

创建 `api.py`：

```python
from flask import Flask, request, jsonify
from flask_cors import CORS
import os
import tempfile
import base64
import numpy as np
from PIL import Image
import io
from inference import SignLanguageRecognizer, Vocabulary

app = Flask(__name__)
CORS(app)  # 允许跨域请求

# 初始化识别器和词汇表
MODEL_PATH = './models/continuous_v4_run2/best_model.pth'
DICT_PATH = './dictionary.txt'

recognizer = SignLanguageRecognizer(model_path=MODEL_PATH, device='cuda')
vocab = Vocabulary(DICT_PATH)


@app.route('/health', methods=['GET'])
def health_check():
    """健康检查接口"""
    return jsonify({'status': 'ok', 'model_loaded': True})


@app.route('/predict', methods=['POST'])
def predict():
    """
    视频文件识别接口
    
    Request:
        - video: 视频文件 (multipart/form-data)
    
    Response:
        {
            'success': True,
            'predictions': [词ID列表],
            'text': '识别文本'
        }
    """
    if 'video' not in request.files:
        return jsonify({'success': False, 'error': 'No video file provided'}), 400
    
    video_file = request.files['video']
    
    # 保存临时文件
    with tempfile.NamedTemporaryFile(delete=False, suffix='.mp4') as tmp_file:
        video_file.save(tmp_file.name)
        tmp_path = tmp_file.name
    
    try:
        # 进行推理
        predictions = recognizer.predict(tmp_path)
        
        # 解码为文本
        text = vocab.decode(predictions)
        
        return jsonify({
            'success': True,
            'predictions': predictions.tolist(),
            'text': text
        })
        
    except Exception as e:
        return jsonify({
            'success': False,
            'error': str(e)
        }), 500
        
    finally:
        # 清理临时文件
        if os.path.exists(tmp_path):
            os.unlink(tmp_path)


@app.route('/predict_frame', methods=['POST'])
def predict_frame():
    """
    单帧实时识别接口（需要维护帧缓冲区）
    
    Request:
        {
            'frame': base64编码的图像数据,
            'session_id': 会话ID（用于维护帧缓冲区）
        }
    
    Response:
        {
            'success': True,
            'predictions': [词ID列表],
            'text': '识别文本',
            'buffer_status': 'full' | 'filling'
        }
    """
    data = request.json
    
    if 'frame' not in data:
        return jsonify({'success': False, 'error': 'No frame data provided'}), 400
    
    try:
        # 解码base64图像
        frame_data = base64.b64decode(data['frame'])
        frame = Image.open(io.BytesIO(frame_data))
        
        # TODO: 实现帧缓冲区管理
        # 这里需要维护一个全局的帧缓冲区，每个session_id对应一个缓冲区
        # 当缓冲区满48帧时进行推理
        
        return jsonify({
            'success': True,
            'message': 'Frame received',
            'buffer_status': 'filling'
        })
        
    except Exception as e:
        return jsonify({
            'success': False,
            'error': str(e)
        }), 500


if __name__ == '__main__':
    # 生产环境建议使用gunicorn
    # gunicorn -w 4 -b 0.0.0.0:5000 api:app
    app.run(host='0.0.0.0', port=5000, debug=False)
```

### 1. 视频上传组件

创建 `VideoUploader.js`：

```jsx
import React, { useState } from 'react';
import axios from 'axios';
import './VideoUploader.css';

const API_URL = 'http://localhost:5000';  // 根据实际部署调整

function VideoUploader() {
  const [file, setFile] = useState(null);
  const [result, setResult] = useState(null);
  const [loading, setLoading] = useState(false);
  const [error, setError] = useState(null);

  const handleFileChange = (event) => {
    const selectedFile = event.target.files[0];
    if (selectedFile && selectedFile.type.startsWith('video/')) {
      setFile(selectedFile);
      setError(null);
    } else {
      setError('请选择有效的视频文件');
    }
  };

  const handleUpload = async () => {
    if (!file) {
      setError('请先选择视频文件');
      return;
    }

    const formData = new FormData();
    formData.append('video', file);

    setLoading(true);
    setError(null);

    try {
      const response = await axios.post(`${API_URL}/predict`, formData, {
        headers: {
          'Content-Type': 'multipart/form-data',
        },
        timeout: 30000,  // 30秒超时
      });

      if (response.data.success) {
        setResult(response.data);
      } else {
        setError(response.data.error || '识别失败');
      }
    } catch (err) {
      setError(err.response?.data?.error || '网络错误，请稍后重试');
    } finally {
      setLoading(false);
    }
  };

  return (
    <div className="video-uploader">
      <h2>手语视频识别</h2>
      
      <div className="upload-section">
        <input
          type="file"
          accept="video/*"
          onChange={handleFileChange}
          disabled={loading}
        />
        <button onClick={handleUpload} disabled={!file || loading}>
          {loading ? '识别中...' : '开始识别'}
        </button>
      </div>

      {error && <div className="error-message">{error}</div>}

      {file && (
        <div className="preview">
          <h3>视频预览：</h3>
          <video width="320" height="240" controls>
            <source src={URL.createObjectURL(file)} type={file.type} />
          </video>
        </div>
      )}

      {result && (
        <div className="result">
          <h3>识别结果：</h3>
          <div className="result-text">{result.text}</div>
          <details>
            <summary>原始预测数据</summary>
            <pre>{JSON.stringify(result.predictions, null, 2)}</pre>
          </details>
        </div>
      )}
    </div>
  );
}

export default VideoUploader;
```

### 2. 实时摄像头组件

创建 `CameraRecognizer.js`：

```jsx
import React, { useRef, useEffect, useState, useCallback } from 'react';
import axios from 'axios';
import './CameraRecognizer.css';

const API_URL = 'http://localhost:5000';
const FRAME_INTERVAL = 100;  // 每100ms捕获一帧
const BUFFER_SIZE = 48;      // 缓冲区大小

function CameraRecognizer() {
  const videoRef = useRef(null);
  const canvasRef = useRef(null);
  const [isRecording, setIsRecording] = useState(false);
  const [result, setResult] = useState(null);
  const [bufferCount, setBufferCount] = useState(0);
  const frameBuffer = useRef([]);
  const intervalRef = useRef(null);

  // 启动摄像头
  const startCamera = async () => {
    try {
      const stream = await navigator.mediaDevices.getUserMedia({ 
        video: { 
          width: 320,
          height: 240,
          frameRate: 10
        } 
      });
      
      if (videoRef.current) {
        videoRef.current.srcObject = stream;
      }
      
      setIsRecording(true);
      startFrameCapture();
    } catch (err) {
      console.error('Camera error:', err);
      alert('无法访问摄像头，请检查权限设置');
    }
  };

  // 停止摄像头
  const stopCamera = () => {
    if (videoRef.current && videoRef.current.srcObject) {
      const tracks = videoRef.current.srcObject.getTracks();
      tracks.forEach(track => track.stop());
      videoRef.current.srcObject = null;
    }
    
    if (intervalRef.current) {
      clearInterval(intervalRef.current);
    }
    
    setIsRecording(false);
    frameBuffer.current = [];
    setBufferCount(0);
  };

  // 捕获帧
  const captureFrame = useCallback(() => {
    const video = videoRef.current;
    const canvas = canvasRef.current;
    
    if (!video || !canvas) return;
    
    const ctx = canvas.getContext('2d');
    canvas.width = 128;  // 模型输入尺寸
    canvas.height = 128;
    
    // 绘制视频帧到canvas
    ctx.drawImage(video, 0, 0, canvas.width, canvas.height);
    
    // 转换为base64
    const frameData = canvas.toDataURL('image/jpeg', 0.8);
    
    // 添加到缓冲区
    frameBuffer.current.push(frameData);
    
    // 保持缓冲区大小
    if (frameBuffer.current.length > BUFFER_SIZE) {
      frameBuffer.current.shift();
    }
    
    setBufferCount(frameBuffer.current.length);
    
    // 当缓冲区满时进行推理
    if (frameBuffer.current.length === BUFFER_SIZE) {
      sendForRecognition();
    }
  }, []);

  // 开始捕获帧
  const startFrameCapture = () => {
    intervalRef.current = setInterval(captureFrame, FRAME_INTERVAL);
  };

  // 发送识别请求
  const sendForRecognition = async () => {
    try {
      // 这里简化处理，实际应该发送所有帧
      // 由于数据量大，可能需要优化传输方式
      const response = await axios.post(`${API_URL}/predict_frame`, {
        frames: frameBuffer.current,
        session_id: 'default'
      });

      if (response.data.success && response.data.text) {
        setResult(response.data);
      }
    } catch (err) {
      console.error('Recognition error:', err);
    }
  };

  useEffect(() => {
    return () => {
      stopCamera();
    };
  }, []);

  return (
    <div className="camera-recognizer">
      <h2>实时手语识别</h2>
      
      <div className="camera-controls">
        {!isRecording ? (
          <button onClick={startCamera} className="start-btn">
            启动摄像头
          </button>
        ) : (
          <button onClick={stopCamera} className="stop-btn">
            停止识别
          </button>
        )}
      </div>

      <div className="camera-preview">
        <video
          ref={videoRef}
          autoPlay
          muted
          playsInline
          width="320"
          height="240"
        />
        <canvas ref={canvasRef} style={{ display: 'none' }} />
      </div>

      {isRecording && (
        <div className="buffer-status">
          缓冲区: {bufferCount}/{BUFFER_SIZE} 帧
          {bufferCount === BUFFER_SIZE && (
            <span className="recognizing"> (识别中...)</span>
          )}
        </div>
      )}

      {result && result.text && (
        <div className="realtime-result">
          <h3>识别结果：</h3>
          <div className="result-text">{result.text}</div>
        </div>
      )}
    </div>
  );
}

export default CameraRecognizer;
```

    # 前端静态文件

        try_files $uri $uri/ /index.html;

    # 后端API代理

## 常见问题

### Q1: 模型推理速度慢怎么办？

**解决方案：**
1. 使用GPU加速
2. 降低视频分辨率（如从128降到96）
3. 减少帧数（如从48降到32）
4. 使用模型量化
5. 使用TensorRT优化

### Q2: 摄像头实时识别延迟高怎么办？

**解决方案：**
1. 降低采样率（每5帧取1帧）
2. 使用滑动窗口而不是等待48帧
3. 降低视频分辨率
4. 使用WebRTC进行低延迟传输

### Q3: 识别准确率低怎么办？

**解决方案：**
1. 确保视频质量良好（光线充足、背景简单）
2. 检查预处理流程是否正确
3. 增加训练数据
4. 调整模型参数（如增加隐藏层维度）
5. 使用数据增强

### Q4: 如何支持更多词汇？

**解决方案：**
1. 扩展词汇表（修改dictionary.txt）
2. 重新训练模型（需要新的训练数据）
3. 使用迁移学习（基于现有模型微调）

### Q5: 如何部署到移动端？

**解决方案：**
1. 将模型转换为ONNX格式
2. 使用TensorFlow Lite或PyTorch Mobile
3. 使用MNN或NCNN等移动端推理框架
4. 降低模型复杂度（使用更小的ResNet）
